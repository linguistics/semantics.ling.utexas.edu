LIWC - Categorical-Dynamic Index

http://lingsa.org/science

This application provides a way to upload PDFs or text documents, or enter text directly, and compute the Categorical-Dynamic Index for the document.

Presidential Speech Temporal Analysis

http://lingsa.org/speeches

This application allows the user to compare the trends of word use within a single president's presidential term, or across several presidents.
